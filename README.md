# Task 7: Basic Sales Summary Using SQLite and Python

## Objective

To extract basic sales insights such as total quantity sold and total revenue using SQL queries within Python and visualize the results using a simple bar chart.

## Dataset Used

Self-created  Sales Dataset stored in a SQLite database (sales_data.db).

## Steps Performed

• Created and connected to a SQLite database using Python

• Designed a sales table containing product, quantity, and price fields

• Inserted company-level sales data into the database

• Executed SQL aggregation queries to calculate total quantity sold and total revenue per product

• Loaded SQL query results into a pandas DataFrame

• Displayed the summarized sales data using print statements

• Visualized revenue by product using a basic matplotlib bar chart

• Saved the generated chart as an image file

• Closed the database connection after completing operations

## Tools Used

• Python

• SQLite (sqlite3)

• Pandas

• Matplotlib

• Jupyter Notebook

## Output

• Printed sales summary table showing total quantity and revenue per product

• Simple bar chart visualizing product-wise revenue

• Saved chart file: sales_chart.png
